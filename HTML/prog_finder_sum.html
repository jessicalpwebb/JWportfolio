UCCS Program Finder Redesign

Improving student user experience through UX design principles

Timeline February 2025 - May 2025
Team Joslyn Bayley, Wander Barnes, Jessica Webb
Role UX Designer
View Figma Prototype →

Project Overview

Our mission was to redesign the UCCS Program Finder to improve students' user experience. The goal was to improve the site using UX Design concepts such as navigation, color, structure, and information architecture. Additionally, we followed UCCS design standards to assist the site's structure while adding our own expertise to display creativity in user experience.

UX Design User Research Prototyping Figma Interaction Design Information Architecture Accessibility

Client Discovery

Before tackling the page's redesign, we met with a client to discuss issues and improvements needed for the page. The following were the main takeaways from our discussion:

Page needs less clutter

Page needs to be more dynamic

Page needs to be "jazzed up"

User Tasks

With these points, our team decided to create 3 main user tasks:

  1. Filter and Search Improvement
  2. Chat and UI Enhancement
  3. New Program Finder Quiz

User Persona

Meet Mia Thompson

We chose to base our persona on a primary audience: incoming students. With that in mind, we created our persona Mia Thompson who is a 19 year old transfer student.

Mia Thompson is an aspiring explorer trying to navigate the challenge of a major life transition. Some of her key qualities are she is from outside the state, looks forward to exploring new opportunities but nervous about the adjustments that come with it, has a strong academic foundation, and relies on digital tools to stay connected.

Other features included moving the feedback link to the bottom of the page since it has a low usage rate and making the page have a friendlier feel. Through peer and class feedback, some improvements we had to include in our prototype included:

  • Fixing the structure of the quiz
  • Change the position of the Chat feature
  • Add more features to make home page and quiz friendlier

UCCS Academic Program Finder - Final Design

Chat with Admissions Feature

For our final prototype, the following were updated for the home page:

  • Updated visual layout and brand imagery
  • Chat has clearer hours and fallback options when offline
  • Chat is positioned in an accessible position

The students should be able to filter programs with ease, see programs they are looking for, and visually be able to navigate features on page and follow all visual queues.

Interactive Quiz Feature

Our program finder quiz involves an interactive and engaging multiple-choice quiz and a results page inspired by personality tests, including subjects in academic interests, personality traits, top course matches, and top department matches.

Additionally, we used UCCS pictures and colors throughout to catch the spirit of the campus. Adjustments in contrast, structure, and navigation were also included to ensure students are excited to join the UCCS family.
